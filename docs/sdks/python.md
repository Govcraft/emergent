# Python SDK

The `emergent-client` package provides the Python SDK for building Sources, Handlers, and Sinks.

## Installation

```bash
pip install emergent-client
```

Or with uv:

```bash
uv add emergent-client
```

- **PyPI**: [emergent-client](https://pypi.org/project/emergent-client/)
